# Rossi Manufacturing

Rossi Manufacturing is a fictional small-to-medium-sized manufacturing company created for this home lab series.

The purpose of this environment is to simulate a realistic enterprise IT infrastructure across multiple projects while demonstrating practical system administration, Active Directory, cloud, and automation skills.

## Current Infrastructure

- Windows Server 2022
- Active Directory Domain Services (AD DS)
- Active Directory DNS
- Windows 11 Workstation
- Organizational Units (OUs)
- Department-based file shares
- NTFS permissions
- Group Policy
- Drive mapping
- Access-Based Enumeration (ABE)

## Planned Expansion

- Microsoft Azure
- Cloud-hosted Active Directory
- PowerShell automation
- Microsoft Entra ID
- Security monitoring
- Enterprise networking
