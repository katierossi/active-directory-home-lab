# Active Directory Home Lab

## Overview

This project documents my Windows Server Active Directory home lab.

The goal of this project is to gain hands-on experience with enterprise Windows administration by building and managing an Active Directory environment from scratch.

## Technologies

- Active Directory Domain Services (AD DS)
- DNS
- VMware Workstation
